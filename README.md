# SmartPost Bot 🤖

SmartPost Bot — это автономный Telegram-бот с искусственным интеллектом для автоматической генерации, редактуры и публикации контента. Бот выступает в роли полноценной AI-редакции: сам ищет информацию по заданной теме, парсит статьи, пишет посты в заданном стиле и предоставляет удобную админ-панель для их проверки и доработки.

## 🌟 Основные возможности

*   **Автономная AI-генерация (`/ai_post <тема>`)**:
    *   **Planner**: Нейросеть анализирует тему и формирует точный поисковый запрос.
    *   **Searcher**: Поиск актуальных новостей и статей через DuckDuckGo (с обходом заблокированных доменов).
    *   **Scraper**: Парсинг содержимого сайтов (извлечение чистого текста).
    *   **Editor**: Написание поста по одному из 6 продуманных шаблонов (Ультракороткий, История, Буллеты, Цифра, До/После, Вопрос-ответ) с учётом культурного контекста.
*   **Интерактивная проверка постов**: Перед публикацией администратор получает пост на проверку и может:
    *   ✅ **Опубликовать** сразу в канал.
    *   ✏️ **Доработать**: Написать комментарий (например, «сделай смешнее» или «убери воду»), и AI перепишет пост с учётом пожеланий.
    *   📁 **В архив**: Отложить пост для публикации в будущем.
    *   🗑 **Удалить**: Забраковать материал.
*   **Удобная система архива (`/drafts`)**: Хранение отложенных постов в локальной SQLite базе данных с возможностью листать страницы (пагинация).
*   **Ручной постинг (`/post`)**: Возможность переслать или написать сообщение боту, чтобы он опубликовал его в канале.
*   **Безопасность**: Доступ к боту и панели управления строго ограничен для `ADMIN_ID`.

## 🛠 Технологии

*   **Python 3.10+**
*   [PyTelegramBotAPI (Telebot)](https://github.com/eternnoir/pyTelegramBotAPI) — интерфейс бота.
*   [OpenRouter API](https://openrouter.ai/) / [Google Gemini](https://aistudio.google.com/) — нейросетевое ядро (генерация, анализ, редактура).
*   [DuckDuckGo Search (DDGS)](https://github.com/deedy5/duckduckgo_search) — поиск информации без лимитов и капчи.
*   [newspaper3k](https://github.com/codelucas/newspaper) — извлечение текста из статей.
*   **SQLite3** — легковесная БД для хранения архива постов.

## 🚀 Установка и запуск

1.  **Склонируйте репозиторий**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/shool.git
    cd shool
    ```

2.  **Установите зависимости**:
    Рекомендуется использовать виртуальное окружение:
    ```bash
    python -m venv venv
    venv\Scripts\activate  # Для Windows
    pip install pyTelegramBotAPI openai duckduckgo-search newspaper3k
    ```

3.  **Настройте конфигурацию (`cfg.py`)**:
    В файле `cfg.py` укажите ваши данные (или создайте `private_cfg.py`, если предусмотрено):
    *   `BOT_TOKEN` — токен вашего бота (от [@BotFather](https://t.me/BotFather)).
    *   `ADMIN_ID` — ваш Telegram ID (чтобы бот слушался только вас).
    *   `CHANNEL_ID` — юзернейм или ID канала (например, `@my_channel`), куда бот будет публиковать посты.
    *   `OPENROUTER_API_KEY` (или `GEMINI_API_KEY`) — ключи доступа к нейросетям.

4.  **Запустите бота**:
    ```bash
    python bot_sender.py
    ```

## 📚 Архитектура проекта

*   `bot_sender.py` — Главный файл. Обработка команд Telegram, интерфейс (инлайн-клавиатуры), взаимодействие с админом.
*   `ai_core.py` — "Мозги" бота. Реализация агентов (Planner, Searcher, Scraper, Editor). Подключение к API нейросети.
*   `db.py` — Работа с базой данных `posts.db`. Функции для сохранения, получения, удаления и подсчёта архивных постов.
*   `cfg.py` — Конфигурационный файл.
*   `portfolio/` — Папка с сайтом-портфолио (HTML/CSS/JS).

## 💡 Использование

После запуска бота, напишите ему в Telegram:
*   `/ai_post Илон Маск` — бот пойдёт в интернет, найдет свежие новости про Маска, сгенерирует посты и пришлёт вам на согласование.
*   Нажмите **"Доработать"** и напишите: *"Добавь шутку про Марс"*, чтобы переписать пост.
*   `/drafts` — открыть список постов, которые вы отложили в архив.
